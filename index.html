Simple photographer CMS

This full-stack MERN application is a Content Management System for building photographer's portfolios. The client-side has the actual portfolio and an admin dashboard accessible through User authentication. The application has Unit Testing with react-testing-library and Jest. Some features are:

  • - Login System with JWT to access the Admin control panel;
  • - Drag and drop system to upload multiple images at a time;
  • - Create and manage new Projects/albums for the images.

FedericoFruzzetti.com

A portfolio website for a photographer client with a focus on images optimization to improve performance and bandwidth consumption. The design is very simple and minimal like he requested. The website is made with Html, SCSS, Javascript, and two external libraries for the carousel and the lightbox gallery. The main focus of the project is the optimization for images and the speed of load, which I achieved with a combination of lazy loadings and srcset tags to serve the right image for the right device.
